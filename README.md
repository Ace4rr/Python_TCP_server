# Python_TCP_server

### Описание
Данный проект представляет собой мини TCP сервер реализованный на Python 
Сервер работает поверх TCP-сокета, принимает сырые HTTP-запросы, парсит их, логирует информацию и формирует корректный HTTP-ответ

### Поддерживаемые возможности 
- Прием HTTP запросов через TCP
- Ручной парсинг запросов
- Формирование HTTP ответа
- Логирование

### Формат ввода 
Сервер принимает любой HTTP-запрос, отправленный через TCP-клиента(удобно через curl, ncat):
`GET /test HTTP/1.1
Host: localhost`

### Алгоритм работы
- Пользователь отправляет HTTP Запрос
- Сервер принимает сырые байты
- Парсер извлекает путь заголовки и тело запроса
- Handler получает данные и формирует ответ
- Сервер собирает HTTP ответ и отдает его клиенту
